# Student Performance Analysis

## 📊 Overview
This project analyzes student performance based on study habits and background factors.

## 🛠 Tools Used
- Python (Pandas)
- Power BI
- Excel

## 📈 Key Insights
- Students who study more hours score higher marks
- Performance comparison across gender
